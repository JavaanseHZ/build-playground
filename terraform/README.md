#Terraform
##Installation
[Donwload newest Version](https://learn.hashicorp.com/terraform/getting-started/install)
Unzip and copy to /usr/local/bin
##Initilizaion of Providers
go to folder with main.tf file (/terraform in repo)
```
terraform init
```
##Plan Deployment
```
terraform plan
```
##Apply Deployment
```
terraform apply
```
##Destroy Deployment
```
terraform destroy
```
